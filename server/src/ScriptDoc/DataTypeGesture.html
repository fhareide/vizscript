Script Reference

Gesture

Description

Holds data of a single gesture event.

Properties

Event As Integer [read-only]Returns the gesture event type (GESTURE_MANIPULATION_START, GESTURE_MANIPULATION_STOP, GESTURE_MOMENTUM_START, GESTURE_MOMENTUM_STOP, GESTURE_TAP, GESTURE_DOUBLE_TAP, GESTURE_TAP_HOLD, GESTURE_TRANSFORMATION, GESTURE_PATTERN).
PatternId As Integer [read-only]Returns the Id of the recognized gesture pattern (PATTERN_INVALID, PATTERN_RIGHT, PATTERN_LEFT, PATTERN_UP, PATTERN_DOWN, PATTERN_CIRCLE, PATTERN_SQUARE, PATTERN_TRIANGLE)
Pressure As Double [read-only]Returns the normalized pressure value in the range 0.0 to 1.0.
Recognizer As GestureRecognizer [read-only]Returns the recognizer which generated this gesture.
TraceCount As Integer [read-only]Returns the number of traces involved in this gesture.
Valid As Boolean [read-only]Returns true if the object is valid.
xy As Vertex [read-only]Returns the gesture position in screen-coordinates.
